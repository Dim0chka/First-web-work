What I use

Service

In this section, I will tell you what services I used to write this site. First, Visual Studio Code. This is a code editor that Microsoft developed. The second, Figma, is a graphic editor for UX UI designers and web developers. And third, Github is a web service that hosts billions of lines of code, and every day millions of programmers from all over the world come together to jointly develop open source IT projects.

Visual Studio Code

In this editor, I worked with programming languages such as html, css and javascript. In Javascript, I used jQuery. jQuery is a cross-platform JavaScript library designed to simplify client-side HTML scripting. jQuery is currently the most popular JavaScript library, which is installed on 65% of the 10 million most visited sites on the Internet.

Figma

In Figma, you can draw interface elements, create an interactive prototype of the site and application, illustrations, and vector graphics. That's why I used Figma as a prototype for my site.

Github

Github is a web-based project hosting service using the git version control system. Git is a console utility for tracking and maintaining the history of file changes in your project. With Git, you can roll back your project to an older version, compare, analyze, or merge your changes to the repository. A repository is a repository of your code and its history of changes. I used Github to host my site.
